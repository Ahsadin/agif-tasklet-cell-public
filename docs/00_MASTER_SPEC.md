# 00 Master Spec

## What we are building

AGIF Tasklet Cell is a sealed offline single-task module system with a strict runner boundary.

## Scope

- Define runner + bundle contracts and governance workflow.
- Build in phased milestones (MVP-0 to MVP-5).
- Enforce deterministic validation and conformance-first behavior.

## Non-goals

- No general chatbot behavior.
- No open-ended autonomous agent loops.
- No cloud-required runtime.

## Canonical rule

`docs/SPEC_vFinal.md` is the canonical design source and overrides all other descriptions if any conflict appears.
