# AGENTS.md (Public Repo Rules)

## Scope

- This repository is the public research snapshot for AGIF Tasklet Cell.
- Keep changes small, issue-scoped, and publication-safe.

## Safety

- Never add private product code or artifacts from `CellPOS` or `agif_full_product_v1`.
- Never commit secrets, tokens, private keys, or generated ZIP release bundles.
